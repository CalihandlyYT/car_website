# Установка зависимостей проекта

## Быстрая установка

```bash
pip install -r requirements.txt
```

## Что установится:

1. **Flask** - веб-фреймворк для работы сайта
2. **Pillow** - библиотека для работы с изображениями (для создания иконок)

## Ручная установка

Если хотите установить пакеты по отдельности:

```bash
pip install Flask
pip install Pillow
```

## Проверка установки

После установки проверьте, что все работает:

```bash
python main.py
```

Сайт должен запуститься на `http://localhost:5000`

## Примечания

- Python 3.7+ требуется
- Все остальные библиотеки (sqlite3, hashlib, datetime и т.д.) встроены в Python
- Pillow опционален, если вы не планируете создавать новые иконки
