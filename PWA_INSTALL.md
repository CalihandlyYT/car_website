# Установка приложения на телефон

Ваш сайт теперь поддерживает установку как Progressive Web App (PWA)!

## Как установить на Android:

1. Откройте сайт в браузере Chrome на Android
2. Появится уведомление "Добавить на главный экран" или "Установить приложение"
3. Нажмите "Установить" или выберите в меню браузера "Добавить на главный экран"
4. Приложение появится на главном экране как обычное приложение

## Как установить на iPhone/iPad:

1. Откройте сайт в браузере Safari на iOS
2. Нажмите кнопку "Поделиться" (квадрат со стрелкой вверх)
3. Выберите "На экран «Домой»"
4. Нажмите "Добавить"
5. Приложение появится на главном экране

## Возможности PWA:

- ✅ Работает как нативное приложение
- ✅ Можно установить на главный экран
- ✅ Работает офлайн (базовые функции)
- ✅ Быстрая загрузка
- ✅ Нет необходимости скачивать из магазина приложений

## Примечания:

- Для лучшей работы рекомендуется использовать HTTPS
- Иконки созданы в формате SVG (можно конвертировать в PNG для лучшей совместимости)
- Service Worker кэширует основные страницы для офлайн работы
